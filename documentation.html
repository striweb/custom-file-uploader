User File Management2 Plugin Documentation

Overview

The User File Management2 plugin for WordPress allows users to upload files, while enabling administrators to view all uploads and manage them with options such as deletion with confirmation. This plugin supports a variety of file types and integrates seamlessly with WordPress to enhance user and admin capabilities related to file management.

Features

  • File Uploads: Users can upload files such as images and PDF documents.
  • Admin Management: Administrators can view a list of all uploaded files, search through them, and delete files with a confirmation step for safety.
  • Custom Upload Directories: Files are organized into custom directories based on the user's ID, ensuring organized storage.
  • Shortcodes Support: The plugin introduces several shortcodes for displaying upload forms, listing user-specific uploads, and more, making it easy to integrate into WordPress pages and posts.
  • File Type and Role Restrictions: Administrators can specify allowed file types and user roles for uploading files, offering flexibility and control over the upload process.

Installation

  1. Download the "User File Management2" plugin.
  2. Navigate to your WordPress admin panel, go to Plugins > Add New, and choose the "Upload Plugin" option.
  3. Select the downloaded plugin file and click "Install Now."
  4. After the installation is complete, click on "Activate Plugin."

Configuration

Upon activation, the plugin automatically adds options for allowed file types and user roles. To configure these settings:

  1. Go to Settings > File Uploader in the WordPress admin menu.
  2. Use the provided fields to specify the allowed file types (e.g., image/jpeg, image/png, application/pdf) and roles (e.g., subscriber, contributor) that can upload files.
  3. Click "Save Changes" to apply your settings.

Usage

For Admins

  • Viewing and Managing Uploads: Navigate to the WordPress admin panel, and use the shortcode [list_all_user_uploads] on any page or post to display a searchable and paginated list of all user uploads.
  • File Deletion: Files can be deleted directly from the list by clicking the delete option next to the relevant file (requires confirmation).

For Users

  • Uploading Files: Users can upload files by filling out the form provided by the [custom_user_upload_form] shortcode on any page or post. The form includes a file selection input and an upload button.
  • Viewing Own Uploads: Users can view their own uploads by visiting the page or post containing the [list_custom_uploads] shortcode. This displays a list of their uploaded files with options to download or view them.

Customization

The plugin allows for some customization, including:

  • Custom File Types and User Roles: Admins can adjust the allowed file types and roles through the plugin's settings page.
  • Styling: The plugin enqueues specific CSS files for admin (admin-style.css) and frontend (frontend-style.css) styling. These can be modified or overridden in your theme's CSS file for custom styling.

Deactivation and Cleanup

Deactivating the plugin will remove the custom options for allowed file types and user roles from the database but will not delete any uploaded files or directories.
